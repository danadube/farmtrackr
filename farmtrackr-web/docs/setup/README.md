# Setup & Configuration Documentation

This directory contains documentation for setting up and configuring the FarmTrackr application.

## Files

- **DATABASE_SETUP.md** - Instructions for setting up the production database (PostgreSQL on Vercel)
- **DEPLOYMENT_NOTES.md** - Deployment instructions and notes for production
- **QUICK_FIX_PRODUCTION.md** - Quick troubleshooting guide for production issues
- **VERSION_UPDATE.md** - Guide for updating version numbers and build information

## Quick Start

For initial setup, start with **DATABASE_SETUP.md** to configure your database connection.

For deployment, see **DEPLOYMENT_NOTES.md**.
