# Setup & Configuration Documentation

This directory contains all setup, configuration, and deployment documentation for the FarmTrackr web application.

## Files

- **DATABASE_SETUP.md** - Database setup and migration instructions
- **DEPLOYMENT_NOTES.md** - Deployment guide and procedures
- **VERCEL_DEPLOYMENT.md** - Vercel deployment configuration
- **VERSION_UPDATE.md** - Version update procedures
- **QUICK_FIX_PRODUCTION.md** - Production troubleshooting quick fixes
- **OPENAI_API_KEY_SETUP.md** - OpenAI API key configuration
- **GOOGLE_CLOUD_SETUP_FARMTRACKR.md** - Google Cloud Console setup guide
- **GIT_REPOSITORY_CORRUPTION_RECOVERY.md** - Git repository corruption recovery procedures

## Quick Start

1. **Initial Setup**: Start with `DATABASE_SETUP.md`
2. **Deployment**: Follow `VERCEL_DEPLOYMENT.md` or `DEPLOYMENT_NOTES.md`
3. **Google Integration**: See `../google/` for Google OAuth and API setup
4. **API Keys**: Configure OpenAI API key using `OPENAI_API_KEY_SETUP.md`
