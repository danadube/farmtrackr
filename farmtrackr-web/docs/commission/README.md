# Commission System Documentation

This directory contains all documentation related to the commission tracking system, calculations, and related features.

## Files

- **COMMISSION_ROADMAP.md** - Commission dashboard product roadmap (from v3.x)
- **COMMISSION_ROADMAP_SUMMARY.md** - Summary of commission roadmap
- **CALCULATION_COMPARISON_ANALYSIS.md** - Analysis comparing calculation methods
- **COMMISSION_IMPROVEMENT_ROADMAP.md** - Roadmap for commission system improvements
- **COMMISSION_PARITY_STATUS.md** - Status of parity between Swift app and web app
- **COMMISSION_SHEET_SCANNER.md** - AI commission sheet scanner documentation
- **COMMISSION_SYSTEM_AUDIT.md** - System audit and analysis
- **COMMISSION_SCANNER_INTEGRATION_SUMMARY.md** - Integration summary for commission scanner
- **COMMISSION_INTEGRATION.md** - Commission integration planning
- **PARITY_ACHIEVED_SUMMARY.md** - Summary of parity achievements

## Note

The main project roadmap is in `/ROADMAP.md` (project root). This directory contains commission-specific planning documents from the earlier commission dashboard development.
