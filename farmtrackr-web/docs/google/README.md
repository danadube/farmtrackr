# Google Integration Documentation

This directory contains all documentation related to Google integrations (OAuth, Sheets, Contacts).

## Files

### OAuth Setup
- **QUICK_GOOGLE_SETUP.md** - Quick start guide (5 minutes)
- **GOOGLE_OAUTH_SETUP.md** - Complete OAuth setup guide
- **WEB_GOOGLE_OAUTH_SETUP.md** - Web app specific OAuth setup
- **OAUTH_CONSENT_SCREEN_SETUP.md** - OAuth consent screen configuration
- **OAUTH_QUICK_REFERENCE.md** - Quick reference for OAuth settings
- **OAUTH_TROUBLESHOOTING.md** - Troubleshooting OAuth issues

### Google Services
- **GOOGLE_SHEETS_SETUP.md** - Google Sheets API setup
- **GOOGLE_CONTACTS_INTEGRATION.md** - Google Contacts integration plan

### Configuration
- **SCOPE_CONFIGURATION.md** - OAuth scope configuration details
- **SECURITY_INCIDENT_RESPONSE.md** - Security incident response procedures

## Quick Start

1. **First Time Setup**: Follow `QUICK_GOOGLE_SETUP.md`
2. **OAuth Issues**: Check `OAUTH_TROUBLESHOOTING.md`
3. **Detailed Setup**: See `GOOGLE_OAUTH_SETUP.md` for complete instructions
