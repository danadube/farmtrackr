## üìê Design Principles

1. **Apple Aesthetic**: Clean, minimal, with subtle shadows and smooth transitions
2. **8-Point Grid System**: All spacing uses multiples of 8px
3. **Gradient Headers**: Green headers use gradient from Meadow Green to Forest Green
4. **Colored Left Borders**: Cards have 4px colored left borders for visual hierarchy
